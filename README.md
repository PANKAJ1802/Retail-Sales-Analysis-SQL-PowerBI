# 📊 Customer Behavior & Sales Insights Dashboard

## 📌 Project Overview
This project demonstrates a complete end-to-end data analytics workflow, simulating real-world business analysis. It focuses on understanding customer behavior, purchase patterns, and revenue distribution using Python, SQL, and Power BI.

The goal of this project is to extract meaningful insights from raw data and present them through an interactive dashboard to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze customer demographics and purchasing behavior
- Identify key revenue-generating categories
- Understand customer segmentation and loyalty patterns
- Provide actionable business insights through visualization

---

## 🛠️ Tools & Technologies Used
- **Python** → Data Cleaning, Transformation, EDA  
- **SQL** → Data Analysis, Business Queries  
- **Power BI** → Dashboard, Visualization, Insights

## 📈 Key Insights
- Clothing category generates the highest revenue among all categories
- ~70% of customers are non-subscribers, indicating growth opportunity in loyalty programs
- Mid-aged and senior customers contribute the most to revenue
- Average purchase value remains consistent around $60
- Sales distribution shows clear variation across product categories

---

## 📂 Project Workflow
1. **Data Preparation (Python)**
   - Cleaned and transformed raw data
   - Handled missing values and inconsistencies
   - Performed exploratory data analysis

2. **Data Analysis (SQL)**
   - Wrote queries to analyze customer segments
   - Extracted insights on revenue and purchasing patterns
   - Simulated business scenarios

3. **Visualization (Power BI)**
   - Built an interactive dashboard
   - Added filters for dynamic analysis
   - Designed KPIs and category-level insights

4. **Reporting**
   - Summarized insights into business recommendations
   - Created a presentation for stakeholders

---

## 💡 Business Recommendations
- Focus marketing efforts on high-performing categories like Clothing
- Introduce subscription benefits to convert non-subscribers
- Target mid-aged and senior customers for higher ROI campaigns
- Optimize low-performing categories through promotions or pricing strategies
